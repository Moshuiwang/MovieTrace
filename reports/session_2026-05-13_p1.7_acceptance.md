## 新增 CLI 命令

```bash
# TMDb 热门源采集
PYTHONPATH=src python -m movietrace.cli fetch-tmdb-trending --date 2026-05-13

# Trakt 热门源采集
PYTHONPATH=src python -m movietrace.cli fetch-trakt-trending --date 2026-05-13

# 本地查阅 content_updates
PYTHONPATH=src python -m movietrace.cli inspect-updates [--days N] [--priority P0,P1] [--format table|json|md]

# 单条详情
PYTHONPATH=src python -m movietrace.cli inspect-updates --id <content_update_id>
```
