# 工作日报规范（已废弃）

> 2026-05-23 起废弃。
> 后续 Agent **不要**创建新的 `journal/` 会话日报。

## 当前规则

- 会话收尾按 [`session-checklist.md`](session-checklist.md) 执行。
- 需要同步当前状态时，只更新 `STATE.md`、任务包、ADR 或 PR 描述。
- `journal/` 目录仅保留历史文件，供追溯旧会话使用。

## 历史说明

该流程已取消，历史 journal 不需要迁移或删除。
