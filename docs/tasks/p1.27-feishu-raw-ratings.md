## 风险点

- **语义切换歧义**：飞书表历史行"IMDb 评分"仍是 0-100 旧值，新值是 1-10，两者共存直到全部被 upsert 覆盖。如果运营在覆盖期间查看，会看到混合量纲。本任务不处理此问题（接受渐进覆盖）。
- **`imdb.rating` 可能为 `"N/A"`**（OMDb 未收录时）：`_to_float_rating` 需容错，返回 None 而非报错。
- **飞书字段 ID**：新增字段 ID 需要先从飞书开发者后台获取或通过 `ensure_table_fields` 创建后读取，不能硬编码猜测。
