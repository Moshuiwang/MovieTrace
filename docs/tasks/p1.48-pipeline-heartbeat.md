## 目标

## 非目标
- ❌ 不接入 Prometheus / Datadog / Logtail 等外部监控系统
- ❌ 不自动发飞书告警（心跳只写文件；告警集成是下一个任务包）

## 具体要求

### 1. `heartbeat.py` 接口

```python
import json, os, time
from datetime import datetime, timezone, timedelta

_HB_PATH = os.path.join(os.path.dirname(__file__), "../../../data/pipeline_heartbeat.json")
_TZ_CN = timezone(timedelta(hours=8))

def ping(step: str, detail: str = "", *, hb_path: str = _HB_PATH) -> None:
    """写入心跳文件。step 格式如 '6/8 OMDb enrichment'，detail 如 '40/169'。"""
    payload = {
        "step": step,
        "detail": detail,
        "ts": datetime.now(_TZ_CN).isoformat(timespec="seconds"),
        "pid": os.getpid(),
        "status": "running",
    }
    try:
        with open(hb_path, "w", encoding="utf-8") as f:
            json.dump(payload, f, ensure_ascii=False)
    except OSError:
        pass  # 心跳写失败不能影响主流程

def done(*, hb_path: str = _HB_PATH) -> None:
    """流程正常结束时调用，将 status 改为 done。"""
    try:
        with open(hb_path, "r+", encoding="utf-8") as f:
            data = json.load(f)
            data["status"] = "done"
            f.seek(0); json.dump(data, f, ensure_ascii=False); f.truncate()
    except OSError:
        pass
```

- `ping()` 写失败静默（`except OSError: pass`）——心跳不能 crash 主流程
- `_HB_PATH` 可被测试覆盖（`hb_path` 参数）

### 4. `scripts/check-pipeline-health.sh`

```bash
#!/usr/bin/env bash
# 用法: check-pipeline-health.sh [max_age_minutes]
# exit 0 = 健康；exit 1 = 超时或文件不存在
MAX_AGE=${1:-20}
HB_FILE="$(dirname "$0")/../data/pipeline_heartbeat.json"

if [[ ! -f "$HB_FILE" ]]; then
    echo "WARN: heartbeat file not found (pipeline never ran?)"
    exit 0  # 不存在不告警，可能还没跑过
fi

STATUS=$(python3 -c "import json,sys; d=json.load(open('$HB_FILE')); print(d.get('status',''))")
if [[ "$STATUS" == "done" ]]; then
    echo "OK: pipeline status=done"
    exit 0
fi

TS=$(python3 -c "import json,sys; d=json.load(open('$HB_FILE')); print(d.get('ts',''))")
AGE_SEC=$(python3 -c "
from datetime import datetime,timezone,timedelta
ts=datetime.fromisoformat('$TS')
now=datetime.now(timezone(timedelta(hours=8)))
print(int((now-ts).total_seconds()))
")

if [[ "$AGE_SEC" -gt $((MAX_AGE * 60)) ]]; then
    STEP=$(python3 -c "import json; print(json.load(open('$HB_FILE')).get('step','?'))")
    echo "ALERT: heartbeat stale ${AGE_SEC}s (>${MAX_AGE}min), last step: $STEP"
    exit 1
fi

DETAIL=$(python3 -c "import json; print(json.load(open('$HB_FILE')).get('detail',''))")
echo "OK: age=${AGE_SEC}s step=$STEP detail=$DETAIL"
exit 0
```

- `_HB_PATH` 相对路径计算依赖 `heartbeat.py` 文件位置——测试时用 `hb_path` 参数覆盖，不依赖真实路径
