### 改动 6 — 新增 tests/test_sync.py

新建 `tests/test_sync.py`，覆盖两个纯函数（无网络调用，无飞书依赖）：

**`_to_epoch_ms`（sync.py:79）测试用例：**
1. 仅日期 `"2026-05-16"` + 默认 TZ（Asia/Shanghai）→ 返回正确 epoch ms（断言用 `datetime.fromtimestamp(result/1000, TZ)` 反算）
2. 日期时间 `"2026-05-16 10:30:45"` + 默认 TZ → 正确 epoch ms
3. 日期时间 + `tz=UTC` → 比 CST 解析多 8 小时（28800000 ms 差）
4. 空字符串 / None → 返回 None
5. 格式不合法 `"not-a-date"` → 返回 None（捕获 ValueError）

**`_derive_content_type`（sync.py:308）测试用例：**
1. `rec={"content_type": "movie"}` → `"movie"`（直接 short-circuit）
2. `rec={"update_type": "new_season"}`（无 content_type）→ `"tv"`
3. `rec={"content_update_id": "discovery:movie:12345:2026-05-16"}` → `"movie"`
4. `rec={"content_update_id": "discovery:tv:67890:2026-05-16"}` → `"tv"`
5. `rec={}` → `"unknown"`

测试需 `PYTHONPATH=src` 运行通过。
