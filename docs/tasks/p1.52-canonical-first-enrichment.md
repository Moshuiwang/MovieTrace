## 来源任务
架构分析：`enrich_with_tmdb_details()` 的 skip 条件只看候选的内存字段（`c.tmdb_data`），不看数据库里已持久化的 `canonical_items`。而 `canonical_items.genres_json` / `title_zh` / `networks_json` 是每次 enrichment 写入的永久存储，已入库剧集的数据几乎不变。结果：每次该剧出现在 trending 里，只要 `api_cache` TTL 过期就重新打 TMDb API，哪怕数据完全没变。

## 根因

```
现在：候选 → skip(tmdb_data 内存字段) → api_cache TTL → API 调用 → canonical_items
正确：候选 → skip(canonical_items.genres_json 非空) → 直接用 DB 数据 → 完成
                                                      ↘ 无数据 → api_cache → API
```

`canonical_items` 是持久存储，命中率随时间趋近 100%（只有新剧首次出现才需 API）。`api_cache` TTL 是临时缓存，TTL 一到必须重打，即便数据没变。

## 目标

在 `enrich_with_tmdb_details()` 循环开头增加一个 **canonical_items 快速跳过检查**：
- 若 `canonical_items` 已有 `genres_json`（非空）→ 跳过 detail + zh-CN 的 API 调用，直接把 DB 里的字段应用到候选对象
- 若 `canonical_items` 已有 `title_zh` 但缺 `genres_json` → 只跳过 zh-CN 调用，仍做 detail
- 若两者都缺 → 走现有路径（api_cache → API）

## 相关上下文
- `canonical_items` 通过 `external_ids` 表与 `tmdb_id` 关联：`SELECT ci.genres_json, ci.title_zh, ci.networks_json FROM canonical_items ci JOIN external_ids ei ON ci.canonical_item_id = ei.canonical_item_id WHERE ei.source='tmdb' AND ei.external_id=?`
- `genres_json` 非空 = detail 调用曾成功（格式 `[{"id":18,"name":"Drama"}]`）
- `title_zh` 非空 = zh-CN 调用曾成功
- 对于电影，`networks_json` 可以为空（不影响判断）
- `MergedCandidate` 对象有 `tmdb_id` 字段可直接用

## 具体要求

### 新增辅助函数

```python
def _load_canonical_enrichment(
    conn: sqlite3.Connection, tmdb_id: str, media_type: str
) -> dict | None:
    """从 canonical_items 读已入库的 enrichment 字段。无记录返回 None。"""
    row = conn.execute(
        """SELECT ci.genres_json, ci.title_zh, ci.overview_zh, ci.networks_json
           FROM canonical_items ci
           JOIN external_ids ei ON ci.canonical_item_id = ei.canonical_item_id
           WHERE ei.source = 'tmdb' AND ei.external_id = ?""",
        (str(tmdb_id),),
    ).fetchone()
    if not row:
        return None
    return {
        "genres_json": row[0],
        "title_zh": row[1],
        "overview_zh": row[2],
        "networks_json": row[3],
    }
```

### 新增 `_apply_canonical_enrichment`

```python
def _apply_canonical_enrichment(c: MergedCandidate, data: dict) -> None:
    """将 canonical_items 读出的字段应用到候选对象。"""
    if data.get("genres_json"):
        c.genres_json = data["genres_json"]
    if data.get("title_zh"):
        c.title_zh = data["title_zh"]
    if data.get("overview_zh"):
        c.overview_zh = data["overview_zh"]
    if data.get("networks_json"):
        c.networks_json = data["networks_json"]
```

（执行前确认 `MergedCandidate` 有这些属性，若没有则只更新 `c.tmdb_data` dict）
