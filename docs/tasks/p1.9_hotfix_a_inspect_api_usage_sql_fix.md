## 相关上下文
当前问题：
```python
where = ""  # 无过滤条件时
# 后续 SQL: f"select count(*) from api_usage_log{where} AND status='success'"
# 结果: "select count(*) from api_usage_log AND status='success'" → 非法
```
带 `--date` 时可用，默认调用直接崩溃。

3. 所有 7 个查询使用相同的 `where` + `params` 构建方式
4. 确保带过滤条件和不带条件都正常
