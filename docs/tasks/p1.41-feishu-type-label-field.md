## 具体要求

### 1. 新增 `_compute_type_labels(rec)` 函数

```python
def _compute_type_labels(rec: dict) -> list[str]:
    """Compute 类型标签 multi-select values: content_type + TMDb genre names."""
    labels: list[str] = []
    ct = rec.get("content_type") or ""
    if ct:
        labels.append(ct)
    genres_raw = rec.get("genres_json")
    if genres_raw:
        try:
            genres = json.loads(genres_raw) if isinstance(genres_raw, str) else genres_raw
            for g in genres:
                name = g.get("name", "") if isinstance(g, dict) else ""
                if name and name not in labels:
                    labels.append(name)
        except (ValueError, TypeError):
            pass
    return labels
```

### 2. 在 `fields_extra` 写入"类型标签"

```python
labels = _compute_type_labels(rec)
if labels:
    fields_extra["类型标签"] = labels
```

### 3. 飞书多选字段写入格式确认

执行前必须先用 curl 验证多选写入格式（直接传 list 还是需要包装为 `[{"text": "..."}]`）：

```bash
# 取一条现有记录 record_id 后手工 curl 更新"类型标签"字段，确认接受的格式
curl -X PUT "https://open.feishu.cn/open-apis/bitable/v1/apps/{app_token}/tables/{table_id}/records/{record_id}" \
  -H "Authorization: Bearer {token}" \
  -H "Content-Type: application/json" \
  -d '{"fields": {"类型标签": ["tv", "Drama"]}}'
```

若 curl 返回 code=0，直接用 `list[str]`；若报错则改为 `[{"text": v} for v in labels]`。
