## 风险点

- **`RunResult` 结构**：若无 `errors` 字段需先调研，不能假设字段名。先 `grep -rn "class RunResult" src/movietrace/` 定位再实现。
- **过滤漏斗长度**：被淘汰候选可能上千，只展示 top 10 + "另有 N 项"，避免 latest.md 过长。
- **抽 `build_im_summary_text`**：抽出时必须确认现有 notify.py 测试全部通过，不破坏 IM 通知格式。
- **scoring.py 循环 import**：若 scoring.py import 了 pipeline 其他模块，直接在 export_writer 里 import scoring.py 可能产生循环 import。先检查，必要时改用延迟 import。
